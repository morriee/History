//형변환

int a = 10.5;  //실수형이라 8바이트인데 데이터타입은 4바이트임.
큰 80평 집에서 살다가 18평집으로 오면 짐을 버려야 함.
이것을 묵시적 형변환이라고 한다. 데이터 손실이 발생할 수 도 있다. 항상 발생하는 것은 아님.
만약 이것이 아주 중요한 우주항로 시스템이라01.000000000000000으로 나뉜다면 손실이 큼
c/c++의 단점은 묵시적 형변환을 허용하는 것이다. 
c++(1983)는 c언어(1975)보다 나중에 태어남.
자바 정식버전 (1995) 태어남. 자바는 묵시적인 형변환을 에러처리함. 
c++은 묵시적인형변환으로 에러가 나도 인정하고 돌아갔지만 언제부터인가는 에러처리됨.


int b = 5;

int a =(int)10.5;		//명시적 형변환.
